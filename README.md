# Banking App

A command-line banking application built with Python and MySQL that allows users to create accounts, manage balances, and track transaction history.

## Features

- User registration and login
- Deposit money into an account
- Withdraw money from an account
- Check current account balance
- View transaction history
- Data stored in a MySQL database
- Input validation and error handling

## Technologies Used
- Python
- MySQL
- SQL Queries
- Git & GitHub
